    # Compiled class file
   *.class

   *.log

   *.ctxt

  # Mobile Tools for Java (J2ME)
  .mtj.tmp/

  # Package Files #
  *.jar
  *.war
  *.nar
  *.ear

  *.tar.gz
  *.rar
• What does fast-forward in Git mean?
  Fast-forward is the action done by git when there is a merge is done for a branch that is ahead of another branch that has been checked out previously.

• Why should you have different branches?
  Different branches ensure that independent teams have clarity on what the features they are working on.
